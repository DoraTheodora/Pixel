helper module¶

helper.niceFormattedLongText(longString: str)¶

Method that formats a long string to multiple rows

Parameters

longString – is the string received from the virtual_assistant class, that needs to be formatatted

helper.remove_polite_words(string: str)¶

Method removing the polite words from a string, as they are not used by the virtual_assistant to process a request

Parameters

string – initial string that neeeds polite words to be removed from it

helper.splitter(numberOfWords: int, longString: str)¶

Method created as a helper for the “niceFormattedLongText(longString:str)”

Parameters

  • numberOfWords – denotes how many words are allowed on a single row

  • longString – is the initial string that needs to be parsed

helper.substring_after(string: str, delimiter: str)¶

Method returning the next 2 words after a key word

Parameters

  • string – is the initial string that needs to be parsed

  • delimiter – is the key word searched in the string
